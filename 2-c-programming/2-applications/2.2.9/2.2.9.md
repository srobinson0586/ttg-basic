# Explain how arguments are passed in C

```c
void functionOne(int n);
void functionTwo(int *n);

int main(){
    int number = 512;
    functionOne(number);
    printf("number: %d", number);

    functionTwo(&number);
    printf("number: %d", number);
    return 0;
    }

void functionOne(int n){
    n = 128;
}

void functionTwo(int *n){
    *n = 256;
}
```

- Explain how `functionOne` differs from `functionTwo`.

    ```text
     functionOne:
     functionTwo:
    ```

- What will be printed?

    ```text
     number:
     number:
    ```
